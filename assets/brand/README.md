# BatonJS Brand

The recommended mark is **Flow B**.

It combines three project ideas:

- **Baton**: the vertical stem is the baton and the anchor for orchestration.
- **Flow**: two looped paths form a geometric `B`, echoing parallel and pipeline execution.
- **CLI + SDK**: the dark compact shape, cyan accent, and restrained geometry are designed for GitHub, npm, docs, terminal-adjacent UI, and SDK documentation.

## Assets

- `batonjs-logo.svg` — full lockup for README headers, docs, websites, and social previews.
- `batonjs-icon.svg` — square app/package icon for favicons, avatars, and compact UI.

Use the icon at small sizes and the full lockup where the wordmark needs to be readable. Keep at least one icon-node width of clear space around the mark.
